# DnD Map

Мобильное Android-приложение для управления тактическими картами в настольных ролевых играх (D&D, Pathfinder и подобных). Позволяет загружать карту поля боя, расставлять на ней фишки персонажей и отслеживать их состояние прямо во время игровой сессии.

---

## Возможности

- Загрузка карты из галереи или по URL
- Управление картой: масштаб, перемещение, блокировка, сброс
- Тактическая сетка с настраиваемым размером клетки
- Круглые фишки с изображением, именем и HP, которые можно перетаскивать по карте
- Локальная библиотека карт и фишек с ленивой подгрузкой

---

## Стек

- **Kotlin** + **Jetpack Compose** (Material 3)
- **MVVM**: `UiViewModel` → `DrawerViewModel`, `SelectViewModel`, `CreateViewModel`
- **Coil 3** — загрузка изображений из файла и по сети (OkHttp)
- **ConstraintLayout для Compose** — раскладка бокового меню
- Хранилище — `filesDir` устройства, папки `/MAP` и `/CHIP`, без базы данных
- **Min SDK 21** (Android 5.0) / **Target SDK 36**

## Руководство пользователя

### Боковое меню

Боковое меню открывается кнопкой в верхнем углу экрана и содержит пять действий:

| Пункт | Действие |
|---|---|
| Выбрать карту | Открыть библиотеку и выбрать карту поля боя |
| Выбрать фишку | Открыть библиотеку и выбрать одну или несколько фишек |
| Создать карту | Добавить новую карту в библиотеку |
| Создать фишку | Добавить новую фишку в библиотеку |
| Включить сетку | Включить / выключить тактическую сетку поверх карты |

---

### Создание карты или фишки

1. Открыть меню → **«Создать карту»** или **«Создать фишку»**.
2. Ввести имя для нового элемента в текстовое поле.
3. Нажать **«Загрузить изображение»** — откроется выбор источника:
   - **Галерея** — стандартный системный пикер файлов Android.
   - **Интернет** — ввести прямую ссылку на изображение (URL), после чего Coil загрузит его асинхронно.
4. После загрузки изображение отображается в виде предпросмотра (круглая рамка).
5. Нажать **«Ок»** — элемент сохраняется в локальное хранилище приложения (`filesDir/MAP/` или `filesDir/CHIP/`) с уникальным UUID-именем файла.

> Кнопка «Загрузить изображение» недоступна, пока поле имени пустое.

---

### Выбор карты

1. Открыть меню → **«Выбрать карту»**.
2. В диалоге отображается список карт из библиотеки с ленивой подгрузкой (порциями по 8 штук при прокрутке вниз).
3. Нажать галочку рядом с нужной картой — выбор **одиночный**: новый выбор снимает предыдущий.
4. Нажать **«Ок»** — карта устанавливается как фон и сразу отображается на экране.

Прямо в списке доступна кнопка удаления (иконка корзины) — удаляет карту из библиотеки без возможности восстановления.

---

### Выбор фишек

1. Открыть меню → **«Выбрать фишку»**.
2. В диалоге отображается список фишек из библиотеки (та же ленивая подгрузка).
3. Для каждой фишки в строке доступны:
   - **Галочка** — выбор/снятие выбора. Выбор **множественный**: можно отметить сразу несколько фишек.
   - **Имя** — редактируемое текстовое поле. Позволяет задать кастомное имя прямо перед выставлением на карту.
   - **Счётчик количества** (кнопки `−` / `+`) — сколько экземпляров этой фишки поставить на карту.
4. Нажать **«Ок»** — для каждой отмеченной фишки на карту добавляется указанное количество токенов. Все они появляются в позиции `(0, 0)` и готовы к перетаскиванию.

Кнопка корзины удаляет фишку из библиотеки.

---

### Управление картой

После выбора карта отображается на весь экран. В левом нижнем углу расположены четыре управляющие кнопки:

| Кнопка | Действие |
|---|---|
| Замок | Включить / выключить перемещение карты. Когда перемещение заблокировано, жесты панорамирования игнорируются — карта не съезжает случайно во время игры. |
| Сброс | Сбросить смещение карты в начальную позицию `(0, 0)`. Масштаб при этом не сбрасывается. |
| Масштаб | Показать / скрыть слайдер масштабирования. Диапазон: от **1× до 4×**, шаг — 100 делений. |
| Сетка | Показать / скрыть слайдер размера клетки сетки. Работает только при включённой сетке. |

**Панорамирование** — зажать и потянуть карту двумя пальцами. Смещение накапливается, карта не «прыгает» в начало при следующем касании.

**Сетка** — при включении через меню поверх карты рисуются горизонтальные и вертикальные линии. Размер клетки настраивается слайдером (диапазон 63–189 dp), линии пересчитываются на лету.

---

### Управление фишкой

Каждая фишка — круглый токен с изображением, именем и числом HP (красный текст). Начальное значение HP равно 0.

**Перетаскивание** — зажать и тащить фишку в любое место карты. Жест обрабатывается независимо от карты.

**Нажатие на изображение фишки** — раскрывает панель управления по бокам:

| Элемент | Расположение | Действие |
|---|---|---|
| Увеличить | Слева сверху | Увеличить размер фишки на 5 dp |
| Уменьшить | Слева снизу | Уменьшить размер фишки на 5 dp |
| Удалить | Сверху над фишкой | Убрать фишку с карты (из библиотеки не удаляется) |
| HP + | Справа сверху | Увеличить HP на 1 |
| HP − | Справа снизу | Уменьшить HP на 1 |

**Нажатие на число HP** — открывает диалог ручного ввода. Можно ввести любое целое число и подтвердить «Ok». Значение применяется только если введённое число является корректным целым и отличается от текущего.

**Размер фишки** автоматически масштабируется вместе с картой: итоговый размер зависит от базового размера фишки, размера клетки сетки и текущего масштаба карты.
